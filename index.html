Dokumentasi API

Panduan integrasi sistem shortlink untuk pengembang.

1 URL Structure

Sistem ini menggunakan parameter query string untuk melakukan redirect. Format dasar URL adalah sebagai berikut:

GET https://snx.biz.id/?s={alias}
  • s : Alias atau kode unik dari link.

2 Implementation Example

Contoh penggunaan jika Anda ingin melakukan redirect via JavaScript di sisi klien menggunakan logika yang sama: 

// Mendapatkan parameter 's' dari URL
const params = new URLSearchParams(window.location.search);
const alias = params.get('s');

if (alias) {
    // Lakukan fetch ke database Anda atau redirect
    // Contoh endpoint imajiner:
    fetch(`https://api.snx.biz.id/v1/links/${alias}`)
        .then(res => res.json())
        .then(data => {
            if(data.url) window.location.href = data.url;
        });
}

3 Error Handling

Code Description
404 Link tidak ditemukan atau alias salah.
301 Redirect permanen ke URL tujuan.
